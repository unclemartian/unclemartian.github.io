# 复制所有搜过结果

1. CTRL + F, Type your search string
1. CTRL + SHIFT + L to select all occurrences found (max. 999)
1. ESC (or close search dialog with top-right X)
1. [Optionally] CTRL + I to select whole lines
1. CTRL + C
1. Open new file
1. CTRL + V
